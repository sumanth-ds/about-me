Interactive Setup Guide

Follow this step-by-step guide to get the chatbot running locally.

Check Your Tools

Before you begin, ensure you have Python installed and can get API keys for the required services.

  • Python 3.8 or newer
  • Active Google Cloud API Key
  • Active Together AI API Key

Install Dependencies

Open your terminal in the project directory and run the following command to install all the necessary Python packages.

pip install streamlit langchain-experimental langchain-core langchain-google-genai langchain-community langchain-text-splitters requests beautifulsoup4 python-dotenv langchain-together
Copied!

Configure API Keys & Run Server

Create a folder named .streamlit in your project root, and inside it, create a file named secrets.toml.

# .streamlit/secrets.toml
google = "YOUR_GOOGLE_API_KEY_HERE"
together = "YOUR_TOGETHER_API_KEY_HERE"
Copied!

Finally, run the Streamlit application. It will be accessible at `http://localhost:8501`.

streamlit run app.py
